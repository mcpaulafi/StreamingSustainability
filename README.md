## README

## Sustainability experiment tool

Course project for DATA12004 Sustainability in Computer and Data Sciences I and II 2026 at University of Helsinki.

## System

Designed for Fuxi-laptop

## Contents

### main.py
Starts program and prompts user for settings and saves results.

Run

    python3 main.py


### experiment.py

Class for experiment values

### measurement.py

Collects measuring time, network, and battery values before and after the experiment duration

### settings.py

Contains fixed options and finds out system options
